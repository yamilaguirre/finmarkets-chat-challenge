# Servidor Socket.io

## Instalación

```bash
npm install
```

## Ejecución

```bash
# Modo desarrollo (con auto-reload)
npm run dev

# Modo producción
npm start
```

## Puerto

El servidor corre en: http://localhost:3001

## Eventos

- `connection`: Se dispara cuando un cliente se conecta
- `message`: Recibe y reenvía mensajes a todos los clientes
- `disconnect`: Se dispara cuando un cliente se desconecta
